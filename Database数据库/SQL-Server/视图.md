# 视图

## 视图的特点
- 虚表，存放几个表的一部分
- 只存定义，不存数据
- 随着基表数据变化而变化

### 创建视图

#### 基于单表的视图
```sql
-- 建立信息系学生的视图
CREATE VIEW IS_Student
AS -- 子查询
SELECT Sno,Sname,Sage
FROM Student
WHERE Sdept='IS';
WITH CHECK OPTION --后面再做修改，自动加上Sdept='IS'
```

