# 过滤数据

## WHERE子句
```sql
SELECT prod_name, prod_price 
FROM Products 
WHERE prod_price = 3.49; 
```
**ORDER BY 应该位于WHERE之后**

## WHERE 子句操作符
```sql
SELECT prod_name, prod_price 
FROM Products 
WHERE prod_price < 10; 

SELECT prod_name, prod_price 
FROM Products 
WHERE prod_price <= 10;

-- 不匹配检查
SELECT vend_id, prod_name 
FROM Products 
WHERE vend_id <> 'DLL01';
```

### 范围值检查 WHERE ___ BETWEEN _ AND _ 
```sql
SELECT prod_name, prod_price 
FROM Products 
WHERE prod_price BETWEEN 5 AND 10;
```

### 空值检查 IS NULL
```sql
SELECT prod_name 
FROM Products 
WHERE prod_price IS NULL;
```
