# 检索数据

## Select语句
```sql
SELECT foodName
FROM Foods
```
## 检索多个列
```sql
SELECT prod_id, prod_name, prod_price
FROM Products;
```

## 检索所以列
```sql
SELECT *
FROM Products;
```

## 检索不同的值
DISTINCT关键字：只返回不同的值 。  
DISTINCT 关键字作用于所有的列，不仅仅是跟在其后的那一列。
```sql
SELECT DISTINCT vend_id
FROM Products;
```

## 限制结果
TOP关键字：限制最多返回多少行
```sql
SELECT TOP 5 prod_name
FROM Products;
```
不同的数据库有不同的语句逻辑

## 注释
```sql
SELECT prod_name -- 这是一条注释 
FROM Products;
```
